//===définie dans leCFGparams
f_param_mission_timer ==> temps avant la fin du safestart (en minutes)


//====variables initialisées par
CoreScripts\common\f_setLocalVars.sqf

Variables locales crées sur la machine
==========hommes + véhicules=======
f_var_units ==> toutes les unités
f_var_units_BLU ==> toutes les unités BLUFOR
f_var_units_RES ==> toutes les unités indé 
f_var_units_OPF ==> toutes unités opfor
f_var_units_CIV ==> toutes les unités civiles

f_var_groups_BLU ==> groupes Blufor
f_var_groups_RES ==> groupes indés
f_var_groups_OPF ==> groupes opfor
f_var_groups_CIV ==> groupes civils
f_var_groups ==> tous les groupes
f_var_groups_players ==> tous les groupes contenant au moins un joueur

f_var_vehicles ==> tous les véhicules
f_var_vehicles_BLU ==> tous les véhicules blufor
f_var_vehicles_RES ==> tous les véhicules indé
f_var_vehicles_OPF ==> tous les véhicules opfor
f_var_vehicles_CIV ==> tous les véhicules civils
==========hommes seuls===========
f_var_men ==> tous les hommes
f_var_men_BLU ==> hommes blufor
f_var_men_RES ==> hommes indé
f_var_men_OPF ==> hommes opfor
f_var_men_CIV ==> hommes civils
f_var_men_players ==> joueurs


====initialisée par MRHroster\roleassign.sqf
displayName ==> Variable locale liée au joueur
nom de role assigné au joueur dans le lobby, avec traitement pour les chefs d'équipe
// ex: player getVariable "displayname";

==========variables publiques initialisées parinitadmindeadcount.sqdf
publicPlayerKilled ==> nombre de joeurs morts (incrémentée par onPlayerKilled.sqf)
publicAllPlayers ==> liste de tous les joeurs (en tant que playableunits) présents au lancement

===========initialisées par onPlayerKilled.sqf
hasDied ==> retourne "true" si le joueur est mort une fois
récupérer via player getVariable "hasDied";
